# 🚌 운전자 근로시간 분석 대시보드

**버전:** v1.0  
**대상:** 대원교통 운행 단말기 데이터 기반 근로시간 분석

---

## ⚡ 실행 방법

```bash
pip install -r requirements.txt
streamlit run app.py
```

## 📂 필요 파일

| 파일 | 형식 | 설명 |
|------|------|------|
| 운행데이터 | xlsx (여러 개) | 운행횟수상세_회사명YYYYMM_.xlsx |
| 인가현황 | xlsx | 년도별_인가현황.xls → xlsx 변환 후 업로드 |
| 지정휴일 | CSV | 운전자별 주 2일 지정휴일 (아래 형식) |

### 지정휴일 CSV 형식
```
운전자,지정휴일1,지정휴일2
홍길동,토,일
김철수,금,토
```
요일: 월/화/수/목/금/토/일

## 🔧 처리 로직

### 근로시간 산출
- **방법:** 회차별 순수 운행시간 합산
- 운행출발일시 ~ 운행종료일시 차이 계산
- **자정 넘김 자동 처리** (예: 22:30 → 00:30 = 2시간)

### 이상값 탐지 기준
| 유형 | 기준 | 원인 |
|------|------|------|
| 단시간 | 10분 미만 | 운전자 OFF 미조작 |
| 장시간 | 인가운행시간 + 60분 초과 | 운전자 OFF 지연 조작 |

### 이상값 보정 (우선순위)
1. 동일 **월·요일·노선·출발±60분** 정상 기록 평균
2. 동일 **월·노선** 평균 (fallback)
3. **인가운행시간** (최후 fallback)

### 만근일수 산정
```
만근일수 = 달력일수 − 지정휴일수 − 비중복공휴일수
비중복공휴일 = 법정공휴일 중 지정휴일 요일이 아닌 날
※ 대체공휴일 포함 (2020~2025 내장)
```

## 📊 탭 구성

| 탭 | 내용 |
|---|---|
| 📊 데이터 현황 | 업로드 파일 요약, 노선별 인가현황 |

## 📌 참고사항
- `000000` 미등록 운전자: 데이터에 포함하여 처리 (추후 역추적 등록)
- 인가현황 파일은 xls → xlsx 변환 후 업로드 (LibreOffice/Excel 사용)
- 5년치(약 180만행) 처리 가능 (`@st.cache_data` 적용)
